# 🎬 YouTube Downloader Desktop App

Simple, fast, and open-source YouTube downloader application for Windows.

Built using Python + yt-dlp with automatic GitHub Actions build system.

## Features
- Download YouTube videos
- Convert to MP3
- Simple GUI
- Auto-build installer
- Lightweight
