# yt-down-app
🚀 Simple YouTube Downloader Desktop App  Open-source desktop application to download YouTube videos and audio easily.  Features: ✔ Modern GUI ✔ MP4 &amp; MP3 Download ✔ Automatic FFmpeg setup ✔ Windows Installer Build ✔ GitHub Actions Automation  Built with Python + yt-dlp.  Contributions are welcome!
